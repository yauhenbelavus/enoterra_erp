# 🚀 Быстрый запуск приложения

## 📋 **Что нужно сделать:**

### **1. Отправить изменения в GitHub**
Используйте любой из способов:
- **VS Code**: `Ctrl+Shift+G` → "..." → "Push"
- **Git Bash**: `git push origin main`
- **GitHub Desktop**: Нажмите "Push origin"

### **2. GitHub Actions автоматически:**
- ✅ Скопирует файлы на сервер
- ✅ Установит зависимости
- ✅ Инициализирует базу данных
- ✅ Запустит приложение

### **3. Проверить результат:**
Откройте в браузере: **https://erp.enoterra.pl**

## 🔧 **Если нужно подключиться к серверу:**

```bash
# Подключиться к серверу
ssh $USER@$REMOTE_HOST

# Перейти в папку
cd /home/$USER/enoterra_erp

# Запустить (если нужно)
./start_production.sh

# Проверить статус
pm2 status
```

## 🌐 **Проверка:**

После успешного деплоя:
- ✅ **https://erp.enoterra.pl** - открывается
- ✅ **PM2 статус** - показывает "online"
- ✅ **Логи** - без ошибок

## 🆘 **Если что-то не работает:**

1. **Проверьте GitHub Actions**: https://github.com/yauhenbelavus/enoterra_erp/actions
2. **Подключитесь к серверу** и проверьте логи: `pm2 logs enoterra_erp`
3. **Перезапустите**: `pm2 restart enoterra_erp`

---

**Теперь приложение будет запускаться автоматически после каждого push в GitHub!**
