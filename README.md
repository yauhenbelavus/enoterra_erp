# Enoterra ERP System

Система управления предприятием с функциями инвентаризации, заказов и генерации PDF.

## 🚀 Автоматический деплой

Система настроена для автоматического развертывания на Hostinger через GitHub Actions.

### Текущий статус деплоя:
- ✅ GitHub Actions настроен
- ✅ SSH ключи добавлены
- ✅ База данных будет создана автоматически на сервере
- ✅ Тестирование SSH подключения
- ✅ Тестирование копирования серверных файлов

## 📦 Установка и запуск

### Локальная разработка:
```bash
npm install
cd server && npm install
npm run dev
```

### Продакшн:
Автоматический деплой через GitHub Actions при push в main ветку.

## 🗄️ База данных

- **Локальная разработка:** `server/enoterra_erp.db`
- **Продакшн:** Автоматически создается на сервере

## 📄 Генерация PDF

Система поддерживает генерацию PDF документов для заказов с настраиваемым дизайном.
