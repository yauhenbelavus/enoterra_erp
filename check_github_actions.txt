Как проверить GitHub Actions:

1. Откройте в браузере:
   https://github.com/yauhenbelavus/enoterra_erp/actions

2. Найдите последний workflow "Deploy to VPS"

3. Кликните на него и проверьте:
   - Все ли шаги выполнились успешно (зеленые галочки)
   - Есть ли красные крестики (ошибки)
   
4. Особенно проверьте шаги:
   - "Build frontend"
   - "Deploy frontend to VPS" 
   - "Deploy server files to VPS"
   - "Setup PM2 and restart"

5. Если есть ошибки, кликните на проблемный шаг и посмотрите логи

ВОЗМОЖНЫЕ ПРОБЛЕМЫ:
- SSH ключи не настроены
- Неправильные пути на сервере
- Ошибки при сборке
- PM2 не запустился

Если GitHub Actions завершился с ошибками, нужно их исправить!
